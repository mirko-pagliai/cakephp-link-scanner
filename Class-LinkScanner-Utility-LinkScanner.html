_recursiveScan()
_recursiveScan(string $url, string|null $referer = null)

Internal method to perform a recursive scan. It recursively repeats the scan for all urls found that have not already been scanned and until the maximum depth is reached.

Events

This method will trigger some events:

  • LinkScanner.foundLinkToBeScanned: will be triggered when other links to be scanned are found;
  • LinkScanner.responseNotOk: will be triggered when a single url is scanned and the response is not ok.
Parameters
string $url

Url to scan

string|null $referer = null

Referer of this url
